# Note-Keeper App  
  
This little project is a note-keeping web app built using __Node.js__ and __React\*__.  
  
Some additional features include:  
* Material UI  
* Axios  
* Express.js  
* MongoDB  
  
_\* Functional components and hooks (useState / useEffect)_
